##### Shortcuts

ESC = Normalmodus
:q = beenden
:q! = verwerfen und beenden
:wq =  speichern und beenden
:w = speichern


##### MODI
Modus          | Wofür                |  Wie rein
--------------------------------------------------
Normal         | Navigation, Befehle  | Esc
Insert         | Text schreiben       | i
Command        | Speichern, Beenden   | :

## Navigation im Normalmodus
| Taste | Richtung |
|-------|----------|
| `h` | ← links |
| `j` | ↓ runter |
| `k` | ↑ hoch |
| `l` | → rechts |
Eselsbrücke: `j` sieht aus wie ein Pfeil nach unten.

### Schneller bewegen
| Taste | Wirkung |
|-------|---------|
| `w` | ein Wort vorwärts |
| `b` | ein Wort zurück |
| `0` | Zeilenanfang |
| `$` | Zeilenende |
| `gg` | Dateianfang |
| `G` | Dateiende |

## Text löschen und ändern
| Taste | Wirkung |
|-------|---------|
| `x` | Zeichen unter Cursor löschen |
| `dd` | ganze Zeile löschen |
| `dw` | Wort löschen (ab Cursor) |
| `d$` | bis Zeilenende löschen |
| `d0` | bis Zeilenanfang löschen |

### Ändern (löscht und geht in Insert-Modus)
| Taste | Wirkung |
|-------|---------|
| `cw` | Wort ändern |
| `cc` | ganze Zeile ändern |
| `c$` | bis Zeilenende ändern |

### Rückgängig machen
| Taste | Wirkung |
|-------|---------|
| `u` | undo (rückgängig) |
| `Ctrl+r` | redo (wiederherstellen) |
