→ Alle folgenden Git-Befehle beziehen sich auf diesen Ordner.

git status → Welche Dateien geändert, neu oder bereit für Commit sind.
git add . → Markiert alle Änderungen für den nächsten Commit. Noch nichts gespeichert, nur ausgewählt.
git commit -m "Update notes" → Erstellt einen neuen Versionsstand (Snapshot).  Änderungen sind jetzt lokal fest gespeichert.
git push → Überträgt die lokalen Commits zu GitHub. → Macht die Änderungen für andere Geräte verfügbar.
git pull
